# thinkphp5-snippets package

ThinkPHP 5 代码提醒插件，专门给记不住命名空间又不想使用助手函数的朋友设计的小工具

支持 ThinkPHP 5 的全部常量，配置以及命名空间类库提醒

暂不支持模板内置标签(懒得做)，助手函数提醒(不想做)

反馈邮箱: chun@engineer.com

2016年5月20日于哈尔滨


## 安装插件
```
apm install thinkphp5-snippets
```
